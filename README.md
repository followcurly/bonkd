## ✨ Features

<table>
<tr>
<td width="50%">

### 🎯 **Smart Text Simplification**
AI-powered content simplification that maintains meaning while making it accessible to everyone.

### ⚡ **3 Bonk Levels**
- **Little Bonk**: Light simplification
- **Bonk**: Balanced simplification  
- **Big Bonk**: Explain like I'm 5

### 🔄 **Toggle View**
Switch between original and bonked text instantly with smooth animations.

</td>
<td width="50%">

### 🎨 **Comic Sans Magic**
Bonked text appears in friendly, readable Comic Sans font for maximum accessibility.

### 📱 **Clean Interface**
Beautiful, modern popup with smooth animations and intuitive controls.

### 🧠 **Smart Detection**
Automatically finds and processes main content while ignoring navigation and ads.

</td>

### 💾 **Additional Features**
- ✅ **Remembers Preferences** - Saves your favorite bonk level per page
- ✅ **Layout Preservation** - Keeps website design completely intact
- ✅ **Error Handling** - Graceful fallbacks if processing fails
- ✅ **Re-bonking** - Change levels and reprocess content instantly
- ✅ **State Management** - Intelligent state handling across page loads

## 🎮 How to Use

<div align="center">

### 1️⃣ Install → 2️⃣ Navigate → 3️⃣ Click → 4️⃣ Choose → 5️⃣ Bonk!

</div>

```
🌐 Navigate to any text-heavy webpage (news, Wikipedia, blogs)
🖱️ Click the Bonkd icon in your toolbar  
🎚️ Choose your bonk level (Little Bonk → Big Bonk)
✨ Click "Simplify Page" and watch the magic happen!
🔄 Toggle between original and bonked text anytime
```

## 🔧 Technical Features

### Core Technologies
- **Chrome Extension Manifest V3** - Latest extension framework
- **Google Gemini AI API** - Advanced text simplification
- **Smart Content Detection** - Intelligent article identification
- **CSS3 Animations** - Smooth, modern UI interactions
- **Chrome Storage API** - Persistent user preferences
- **Promise-based Architecture** - Robust state management

### Architecture Highlights
```
📱 Popup Interface (HTML/CSS/JS)
    ↓
🎛️ State Manager (Centralized state handling)
    ↓
🔄 Background Script (AI processing)
    ↓
📄 Content Script (Page content detection)
    ↓
🤖 Gemini AI API (Text simplification)
```

## 🛠️ For Developers

### Project Structure
```
bonkd/
├── 📄 manifest.json         # Extension configuration
├── 🎨 popup.html/css/js     # User interface
├── 🔄 background.js         # AI processing & API communication
├── 📝 content.js            # Content detection & replacement
├── ⚙️ config.js             # Configuration (API key)
├── 🖼️ icons/               # Extension icons (16-128px)
├── 📚 README.md             # This file
├── 🚀 SETUP.md              # Developer setup guide
└── 🛡️ .gitignore           # Git ignore rules
```

3. **Make** your improvements
4. **Test** thoroughly on various websites
5. **Submit** a pull request

### Key Files to Understand

- **`content.js`** - Smart content detection and layout preservation
- **`background.js`** - AI processing with robust error handling
